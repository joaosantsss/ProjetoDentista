SmileCare Odontologia
Descrição do Projeto

O SmileCare Odontologia é um site institucional fictício desenvolvido para a disciplina de Introdução a Websites.

O objetivo do projeto foi criar uma página moderna e responsiva para um consultório odontológico, apresentando informações sobre a clínica, serviços oferecidos, equipe profissional e um sistema simples de agendamento de consultas.

Funcionalidades
Página inicial com banner da clínica
Apresentação dos serviços odontológicos
Informações sobre a equipe
Formulário para agendamento de consultas
Integração com WhatsApp para envio das informações do agendamento
Layout responsivo para dispositivos móveis
